DWG reader

Плагин для загрузки файлов в формате DWG в Топоматик 360

Установить плагин

О плагине

DWG reader позволяет импортировать файлы формата DWG (AutoCAD Drawing) в программу Топоматик 360.

Возможности:

  • Чтение файлов формата .dwg
  • Поддержка различных версий AutoCAD
  • Импорт геометрии и атрибутов

Установка:

  1. Нажмите кнопку "Установить плагин" выше
  2. Подтвердите установку в Топоматик 360
  3. Плагин будет доступен в меню импорта

Использование:

После установки плагин будет доступен при импорте файлов с расширением .dwg

Разработка

Исходный код доступен на GitHub

Локальная разработка:

npm install
npm run serve

Сборка:

npm run build
